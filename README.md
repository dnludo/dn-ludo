# 🎲 DN Ludo

DN Ludo is a classic multiplayer Ludo game developed by HK Games & Co.

## Features

- 2, 3, and 4 Player Mode
- Pass & Play Gameplay
- Modern Mobile-Friendly UI
- Dice Roll Animation
- Smooth Token Movement
- Winner Detection
- GitHub Pages Deployment

## Play

Choose the number of players, enter player names, and start the game.

## Tech Stack

- HTML
- CSS
- JavaScript
- GitHub Pages
